# ВантузМастер — магазин сантехнических инструментов

## Описание

Landing page для продажи вантузов и сантехнических инструментов.

## Функциональность

- Каталог товаров с ценами
- Преимущества компании
- Форма обратной связи
- Адаптивный дизайн

## Технологии

- HTML5
- CSS3
- Flexbox/Grid

## Структура файлов

- index.html
- style.css

## Как использовать

Откройте файл `index.html` в любом современном браузере.
